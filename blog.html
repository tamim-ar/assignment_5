What is the DOM?

What is the purpose of the DOM?

The Document Object Model is an application programming interface (API) for HTML and XML documents (DOM). It outlines a document's logical structure, access, and handling procedures.

How to select an element from the document.

There are six ways to choose an element from the document using JavaScript.

  • 1. getElementById – Search element by element_id.
  • 2. getElementsByTagName – Search element by tag name
  • 3. getElementsByClassName – Search element by class name.
  • 4. getElementsByName – Search element by name attribute.
  • 5. querySelector - Returns the first element that matches the passed selector.
  • 6. querySelectorAll - This method returns all the elements that match the passed selector.

What is Event Bubbling?

Event bubbling is a notion found in the DOM (Document Object Model). After an element gets an event, the event bubbles up (or you may say is communicated or propagated) to the element's parent and grandparent elements in the DOM tree until it reaches the root element.

What is callback function?

Why use callback function?

A JavaScript callback is a function that runs following the conclusion of another function. According to a stricter definition, a callback function is any function that accepts input from another function and uses that input to perform that other function.


Due to the fact that many JavaScript activities are asynchronous, callback methods are required. Instead, it runs in the background when the rest of the code is being executed. The function of a callback is to run code in response to an event.
